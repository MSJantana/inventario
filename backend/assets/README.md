# Assets de Relatórios (PDF)

Coloque aqui as imagens de logo utilizadas nos relatórios em PDF.

Recomendação de arquivos:
- `logo-header.png`: logo principal para o cabeçalho.
- `logo-footer.png`: logo menor para o rodapé.

Tamanhos sugeridos (podem ser ajustados):
- Cabeçalho: largura ~120–160px.
- Rodapé: largura ~70–100px.

Configuração via .env:
- `REPORT_LOGO_PATH=c:\\Temp\\Inventario\\backend\\assets\\logo-header.png`
- `REPORT_LOGO_FOOTER_PATH=c:\\Temp\\Inventario\\backend\\assets\\logo-footer.png`

Caso `REPORT_LOGO_FOOTER_PATH` não seja definido, o sistema usará o mesmo logo do cabeçalho no rodapé.
